信息课专属教程 —— PCL2 启动与联机指南

提示：本网站由 AI (Gemini 3 Pro) 根据提供的安装方法生成

1 下载与放置 PCL2 启动器

  1. 访问蓝奏云链接：https://ltcat.lanzouv.com/b0aj6gsid
  2. 输入密码：pcl2 进入文件夹。
  3. 点击 “PCL 正式版”，然后点击 “普通下载”。
  4. 下载完成后打开这个压缩包，把里面的 Plain Craft Launcher 2.exe 文件复制到你的 D 盘。

2 安装 Java 运行环境（可能是非必须的，但是后续要是遇到问题，可以把这一步做了再试试，没准就成功了）

游玩我们需要用到 Java 环境，请按照以下步骤安装：

  1. 访问此官方链接下载 Java 21 安装包：
    https://download.oracle.com/java/21/latest/jdk-21_windows-x64_bin.exe
  2. 下载好之后，直接双击打开该安装包。
  3. 不需要修改任何设置，一路点击 “下一步” 直到完成安装即可。

3 下载游戏本体并启动

  1. Java 安装好之后，打开 D 盘，双击运行 Plain Craft Launcher 2.exe。
  2. 在启动器顶部的 “下载” 页面中，找到并下载 1.18.2 版本（机房电脑配置拉，带不动更高的版本）。等待进度条走完。
  3. 回到启动器主页，在账号区域输入你的 用户名，注意不能跟其他人的重了。
  4. 点击右下角的 “启动” 按钮，进入游戏！

4 畅玩与机房联机

启动游戏后，你可以选择：

  • 单人游戏： 自己创建一个世界独自探索。
  • 多人游戏： 实现同机房内的局域网联机！
    （联机小贴士：如果你想当房主，进入单人游戏后按 Esc 选择 “对局域网开放”。其他同学就可以在“多人游戏”列表中看到并加入你的世界啦！）

进入世界后建议去设置 -> 视频设置里把画质等调低避免卡顿。
